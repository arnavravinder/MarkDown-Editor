Markdown FAQs

  • What is Markdown? Markdown is a lightweight markup language for creating formatted text using a plain-text editor.
  • How do I create a heading? Use the # symbol followed by a space and the heading text. For example: # Heading 1
  • How do I create a list? Use dashes (-) or asterisks (*) followed by a space and the list item text.
  • How do I create a link? Use the following syntax: [Link text](URL)
  • How do I create bold text? Wrap the text with two asterisks: **bold text**
